# Curiosity Journey

## 소개

**Curiosity Journey**는 아이의 관심사를 연령에 맞는 프로젝트형 학습으로 확장해 주는 AI 기반 수업용 웹앱 프로토타입입니다.

부모나 교사가 아이의 관심사(예: 전기차, 곤충, 공룡, 우주, 하키 등)를 입력하고 나이대를 선택하면, AI가 발달 수준에 맞는 **3단계 성장 여정(발견하기 → 탐험하기 → 창조하기)**을 제안합니다.

이 웹앱은 HTML, CSS, JavaScript만으로 구현한 단일 페이지(MVP) 프로젝트이며, 수업에서 GitHub와 웹앱 배포를 연습하기 위한 예제입니다.

---

## 주요 기능

### 🧠 1. 관심사 + 나이 기반 성장 여정 생성
- 아이의 관심사와 나이대를 입력하면 AI가 맞춤형 프로젝트를 생성합니다.
- 같은 관심사라도 연령에 따라 다른 활동과 탐구 방향을 제공합니다.

### 🌱 2. 3단계 성장 프로젝트
각 관심사를 다음과 같은 성장 단계로 제공합니다.

- 🌱 발견하기
- 🔍 탐험하기
- 🚀 창조하기

각 단계에는 다음 내용이 포함됩니다.

- 목표
- 준비물
- 활동 예시
- 단계별 하브루타 질문
- 부모 지도 Tip

### 🌍 3. 영어 및 융합 탐구
프로젝트와 함께

- 영어 핵심 키워드
- 쉬운 영어 설명
- 과학, 수학, 미술, 사회 등과 연결되는 융합 확장 탐구

를 함께 제공합니다.

---

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 실행합니다.
2. 아이의 관심사를 입력합니다.
3. 아이의 나이대를 선택합니다.
4. **성장 여정 만들기** 버튼을 클릭합니다.
5. 생성된 결과를 확인합니다.

생성되는 결과

- 🧠 관심사 분석
- 🌱 발견하기
- 🔍 탐험하기
- 🚀 창조하기
- 🇬🇧 영어 키워드
- 🌍 융합 확장 탐구

---

## 수업용 주의사항

- 이 프로젝트는 **2회차 수업에서 제작한 단일 HTML 웹앱 프로토타입**입니다.
- 배포 파일은 `index.html` 하나로 구성됩니다.
- 실제 개인정보는 입력하지 않습니다.
- 샘플 데이터만 사용하여 테스트합니다.
- 현재는 화면 동작과 UI를 확인하기 위한 프로토타입이며, 실제 AI API는 연결되어 있지 않습니다.
- 실제 서비스로 발전하려면 다음과 같은 기능이 추가로 필요합니다.
  - 사용자 인증(Login)
  - 데이터 저장(Database)
  - AI API 연동
  - 보안(Security)
  - 권한 관리(Authentication & Authorization)
  - 입력값 검증 및 예외 처리

---

### 프로젝트 목표

이 프로젝트의 목표는 단순히 활동을 추천하는 것이 아니라, **아이의 관심사를 발달 수준에 맞는 프로젝트 학습으로 확장**하여 부모와 아이가 함께 탐구하고 성장할 수 있는 경험을 제공하는 것입니다.
